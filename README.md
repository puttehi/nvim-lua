# personal nvim cfg

Time to learn neovim. Based on ThePrimeagen cfg.

## Prerequisites

Install [`rg`](https://github.com/BurntSushi/ripgrep) and `fd-find`.

## Setup

1. `nvim lua/first-time-setup/packer.lua`
2. `:so`
3. `:PackerSync`
4. `:q`
5. `nvim lua/puttehi/packer.lua`
6. `:so`
7. `:PackerSync`
8. `:q`
